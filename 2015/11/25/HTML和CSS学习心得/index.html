HTML和CSS学习心得

接触HTML和CSS也有一个多月了，大多数标签都已经能够熟练使用，偶尔还会出现一些错误，刚学的时候，大部分代码都要通过HTML和CSS敲出来，自从学了JS，发现HTML和CSS里只需要写一个样式，效果都是靠JS来写，然后就发现HTML和CSS简单了许多。
标签分类：
行级标签：a span img
块级标签：body div p table h1~h6 ol ul dl(字典列表或者自定义列表) form
默认值：
div 默认宽度遵从父级宽度，没有默认高度，内容可以撑开
body 可以设置边框背景，设置边框可以修改width、height值，设置背景会撑满屏幕，不能修改宽高，设置width、height可以修改body 内容，不会影响背景。
body 浏览器的可视区域
a标签：取消下划线text-decoration:none;
ul、ol列表样式类型：list-style-type:none;
图片标签：

html基本样式

<!doctype html>
<html lang="en">
    <head>
        <meta charset="utf-8"/>
        <title></title>
        <link rel="stylesheet" type="text/css" href="index.css"/>
    </head>
    <body>
    </body>
</html>

css三种引入方式

1.外部引入(常用)：<link rel="stylesheet" type="text/css" href="index.css"/>
2.行间样式：<div style=""></div>
3.style标签引入：在</head>上添加<style></style>
